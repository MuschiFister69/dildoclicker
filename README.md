# Dildo Clicker

An incremental clicker game built with vanilla HTML/CSS/JavaScript.

## Play

Open `index.html` in any modern browser. No server or build step required.

Or run with Electron:
```
npm install
npm start
```

## How to Play

- **Stroke** the dildo by clicking/tapping and dragging up and down to earn 💎 Bucks
- **Buy Buildings** for passive income (Bucks per second)
- **Buy Stroke Upgrades** to increase Bucks per stroke
- **Unlock Perks** for global and specialized multipliers
- **Rebirth** — reset buildings for Pleasure Essence (permanent production boost)
- **Ascend** — reset rebirths for Transcendence Points (permanent upgrades)
- **Gamble** at the casino to unlock dildo colors

## Features

- 40 buildings, 40 stroke upgrades, 37 perks, 5 ascension upgrades
- 12 unlockable dildo colors via the casino
- 13 evolution stages — the dildo grows as you earn more
- Combo system for extra Bucks per stroke
- Sound effects (toggleable)
- Auto-save to browser localStorage
- Touch and keyboard support (Spacebar to stroke)

## Rebirth & Ascension

| Mechanic | Resets | Keeps | Gain |
|----------|--------|-------|------|
| Rebirth | Buildings | Click upgrades, Perks, Stroke power, Total Bucks | Pleasure Essence (+50% production each) |
| Ascension | Buildings, Rebirths, PE | Click upgrades, Perks, Stroke power, Total Bucks, Ascension upgrades | Transcendence Points (permanent multipliers) |
